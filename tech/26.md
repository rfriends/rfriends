## headlessブラウザlightpandaをインストールする。  
  
初版 2026/04/14 

```
AI による概要
Lightpandaは、AIエージェントやWeb自動化のために開発された、超高速・超軽量なオープンソースのヘッドレスブラウザです。 
従来のChromium（Chromeのベース）やWebKitの派生ではなく、Zigプログラミング言語を用いてゼロから構築された新しいブラウザエンジンです。
Chromeと比較して約9倍高速で、メモリ使用量は16分の1に抑えられています。
```
  
今回はrfriends3環境に、lightpandaをインストールする方法を記述します。  
  
  
### １．要件  
  
・64bit OSを使用していること。  
・macOS(シリコン、x64)またはLinux(aarch64,x64)を使用していること。  
・rfriends3 ver 1.4.2以降。  

> [!NOTE]  
> Raspberry Pi の場合は、以下の機種で64bit版OS使用のものになります。  
> Raspberry Pi 5/4/3  
> Raspberry Pi Zero 2    
    
### ２．lightpandaのダウンロード  

#### 2.1 ダウンロード  
  
環境に合わせて以下のものをダウンロードしてください。  
  
- Linux (arm)  
  curl -L -o lightpanda-aarch64-linux https://github.com/lightpanda-io/browser/releases/download/nightly/lightpanda-aarch64-linux  
  
- Linux (x64)  
  curl -L -o lightpanda-x86_64-macos https://github.com/lightpanda-io/browser/releases/download/nightly/lightpanda-x86_64-macos  
  
- macOS (シリコン)  
  curl -L -o lightpanda-aarch64-macos https://github.com/lightpanda-io/browser/releases/download/nightly/lightpanda-aarch64-macos  
  
- macOS (x64)  
  curl -L -o lightpanda-x86_64-linux https://github.com/lightpanda-io/browser/releases/download/nightly/lightpanda-x86_64-linux  
  
#### 2.2 インストール  
    
ダウンロードが完了したら、実行権をつけてパスのとおっている場所にlightpandaとしてコピーしてください。  
ここでは、 Linux (arm)版を  /usr/bin/lightpandaにコピーしています。  
  
```
$ chmod a+x ./lightpanda-aarch64
$ sudo cp lightpanda-aarch64 /usr/bin/lightpanda
```
  
#### 2.3 確認  

任意の場所で  
```
$ lightpanda help
1.0.0-nightly.5543+e6cffaef
```
と出力されたら成功です。（バージョンはその時々で異なる）
